# ReversiFall
Puzzle game that combines Reversi and Tetris.

テトリスにオセロ・五目並べのルールを取り入れた、新しい落ち物パズルゲームです。

* 落下テトロミノ固定時に別の色に挟まれているストーンを反転
* 縦・横・斜めのラインで5つ以上同じ色が繋がるとライン消去


## 操作
←：ミノ左移動

→：ミノ右移動

↑：ハードドロップ

↓：高速落下

Z：ミノ右回転
